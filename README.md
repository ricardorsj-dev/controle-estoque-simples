# 📦 Sistema de Controle de Estoque (Python)

Projeto simples de controle de estoque feito em Python, utilizando lista e dicionários.

## 🚀 Funcionalidades

- Adicionar produtos
- Remover produtos
- Listar produtos
- Alerta de estoque baixo

## 🛠 Tecnologias

- Python 3

Estruturas de dados (listas e dicionários)
Modularização de código
Tratamento de erros
Boas práticas em Python
